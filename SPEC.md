# The Goal
To create the game mastermind in react/next.js.


## Requirements
- local game that one person is playing against the computer
- randomized codes / generated by the game
- simple graphics/UI
- button to play again
- if game win no more guesses are needed


## game rules
- there are 6 colors not black or white
- a code is a sequence of 4 colors that can be repeated
- feedback is a sequence of black and white pips sorted by black first then white
- black indicates correct color in a correct spot
- white is correct color but incorrect spot
- feedback only appears after the guess is submitted
- there are 10 guesses before game over
- you win the game if you guess the 4 colors in the correct order


## UI
- opening on start game button
- when start game is pressed then user is taken to an empty game board
- Game board dimensions are 10 by 4. a row for each guess and a column for each color for the guess
- pips are next to each guess row in a 2 by 2
- guess input interface is clicking a colored dot with the ability to submit a full 4 color guess or undo a color choice in sequence order (so if i hit undo it only undoes the most recently added color)
- when you win, a dialog appears that says "you win!" and presents a "play again" button or a "go home" button
